# Contribution Guide

## How Can You Contribute?

1. **Fork** this repository (button at the top right).
2. Download it to your computer (`git clone ...`).
3. Create a new **Branch** (`git checkout -b new-feature`).
4. Make your changes and update the Turkish documentation.
5. Push your changes (`git push`).
6. Open a **Pull Request (PR)**.

> **Note:** Please only edit the Turkish files (`.md`). English translations (`translations/en/`) will be automatically generated by GitHub Actions.
